color — autopy module for converting color formats¶

This module provides routines for converting between the color format (hexadecimal) used by autopy methods and other, more readable, formats (e.g., RGB tuples).

Functions¶

color.rgb_to_hex(r, g, b)¶

Returns hexadecimal value of given RGB tuple. r, g, and b should be in the range 0 - 255.

color.hex_to_rgb(hex)¶

Returns a tuple (r, g, b) of the RGB integer values equivalent to the given RGB hexadecimal value. r, g, and b are in the range 0 - 255.
