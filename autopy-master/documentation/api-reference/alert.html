alert — autopy module for displaying alerts¶

This module contains functions for displaying cross-platform alert dialogs.

Functions¶

alert.alert(msg, title="AutoPy Alert", default_button="OK", [cancel_button])¶

Displays alert with the given attributes. If cancel_button is not given, only the default button is displayed. Returns True if the default button was pressed, or False if cancelled. Note that the arguments are keywords, and can be passed as named parameters (e.g., alert(msg="bar", title="foo")).

Due to limitations in the Win32 API, Windows currently replaces default_button with "OK" and cancel_button (if given) with "Cancel". This may be fixed in a later release.
